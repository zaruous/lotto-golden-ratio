## 자세한 설명 (4,000자 이내)

```
LOTTO GOLDEN RATIO는 역대 로또 6/45 당첨 데이터를 다차원 통계 지표로 분석하는 도구입니다.

⚠️ 본 앱은 실제 복권 구매, 베팅, 현금 거래 기능을 제공하지 않습니다. 통계 분석과 시뮬레이션만 제공하는 정보 도구이며, 어떠한 당첨도 예측하거나 보장하지 않습니다.

■ 주요 기능

① 실시간 생성 & 분석
번호판에서 직접 번호 6개를 선택하면 산술적 복잡도(AC), 홀짝·고저 균형, 소수 출현, 구간 분산 등 통계 규칙 만족 여부와 조화 점수를 실시간으로 확인할 수 있습니다. 통계 규칙을 타깃으로 하는 고속 몬테카를로 스마트 필터 추출기도 제공합니다.

② 역대 전체 대시보드
200회차부터 최근 회차까지 누적 데이터를 전수 분석합니다. 핵심 통계 필터별 합격률 인포그래픽, 조화 등급 분포 차트, 복합 다중 필터링 검색을 제공합니다.

③ 회차별 상세 분석
특정 회차의 실제 당첨 조합이 통계적 관점에서 어떤 규칙을 만족했는지, 하모니 점수(0~100점)와 직전 회차 대비 이월수·이웃수를 함께 분석합니다.

④ 번호 빈도 히트맵
선택한 회차 범위에서 1~45 번호의 출현 빈도와 미출현 기간을 히트맵으로 비교합니다. 최다 출현·최소 출현·최장 미출현 번호를 한눈에 확인할 수 있습니다.

■ 이런 분께 추천합니다
- 로또 번호를 고를 때 감이 아니라 역대 데이터의 통계적 경향을 참고하고 싶은 분
- 내가 고른 조합이 역대 당첨 이력과 비교해 얼마나 "전형적"인지 궁금한 분
- 번호 빈도, 이월 패턴 등 로또 데이터 자체를 탐구하고 싶은 분

■ 특징
- 매주 당첨 결과 자동 갱신
- 오프라인에서도 동작 (내장 데이터 사용)
- 회원가입 불필요, 광고 없음, 개인정보 수집 없음

■ 유의사항
로또 추첨은 매회 독립적인 무작위 추출(약 814만분의 1)이며, 어떠한 통계 기법도 당첨 확률 자체를 높일 수 없습니다. 본 앱은 극단적으로 비정형적인 조합을 걸러내는 통계적 참고 도구일 뿐입니다. 과도한 복권 구매는 가계에 부담이 됩니다.
```

## English (선택 — 영어 등록정보 추가 시)

**App name**: `LOTTO GOLDEN RATIO - Lotto Stats`

**Short description (80 chars)**:
```
Statistical analysis & simulation for Korean Lotto 6/45. No purchase, no betting.
```

**Full description**:
```
LOTTO GOLDEN RATIO analyzes historical Korean Lotto 6/45 winning data with multi-dimensional statistical metrics.

⚠️ This app does NOT offer real lottery purchases, betting, or any real-money transactions. It is a pure statistics and simulation tool and does not predict or guarantee any winnings.

FEATURES
- Real-time analysis: pick 6 numbers and instantly check statistical rule compliance (arithmetic complexity, odd/even & high/low balance, prime counts, section spread) with a harmony score. Includes a fast Monte Carlo smart-filter generator.
- Full history dashboard: pass-rate infographics, harmony grade distribution, and multi-filter search across all historical draws.
- Per-draw deep dive: see which statistical rules each winning combination satisfied, with carry-over/neighbor analysis.
- Frequency heatmap: appearance frequency and drought length for numbers 1-45 over any draw range.

NOTES
Each lottery draw is an independent random event (~1 in 8.14 million). No statistical technique can improve the odds themselves. This app is a reference tool that filters statistically atypical combinations. No sign-up, no ads, no personal data collection. Works offline with bundled data.
```
